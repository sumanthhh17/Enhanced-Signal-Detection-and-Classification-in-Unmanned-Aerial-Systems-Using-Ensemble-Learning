This project focuses on developing an Enhanced Signal Detection and Classification System for Unmanned Aerial Systems (UAS) using advanced machine learning and ensemble learning techniques. The main objective is to improve how UAV (drone) signals are analyzed, classified, and interpreted through robust data-driven models.
Using a comprehensive dataset of RF and system telemetry logs, the project performs extensive data preprocessing, feature engineering, and exploratory data analysis (EDA) to clean, encode, and extract meaningful patterns from the raw data. A custom Battery Level feature is engineered to approximate remaining power based on system usage factors. The cleaned dataset is then used to train and evaluate multiple machine learning models for classification of device status and regression-based prediction of battery level.
For classification, models such as Random Forest, Gradient Boosting, and XGBoost are trained to accurately classify device states, and an ensemble voting classifier combines their strengths to improve performance and stability. For regression, similar models are applied to predict battery levels, with a combined ensemble strategy to maximize predictive accuracy.
The project also includes comprehensive visualizations like distribution plots, heatmaps, confusion matrices, and feature importance charts to provide insights into model behavior and key factors influencing performance. Finally, the best-performing models are saved for deployment in real-world scenarios.
Overall, this work emphasizes the value of ensemble learning, feature engineering, and interpretable analytics in designing reliable signal detection and classification systems for UAV applications — helping enhance situational awareness, system health monitoring, and intelligent decision-making for airborne systems.
